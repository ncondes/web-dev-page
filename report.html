Accessibility

State 3 ways in which your site is accessible.

  1. Responsive Design for Universal Access
    The website is meticulously designed to be fully responsive, ensuring accessibility across a wide array of devices including desktops, tablets, and smartphones. This adaptability guarantees that all users, regardless of their chosen device, can experience the site in the best possible manner, making the foundation's message universally accessible.
  2. Visual Accessibility for Enhanced Readability
    Visual accessibility features are a cornerstone of the site's design. Every image is accompanied by descriptive "alt" text, facilitating content understanding for users employing screen readers. The color palette and font sizes are selected to support users with visual impairments, including those with color vision deficiencies, enhancing readability and ensuring a broader inclusivity.
  3. Audio Accessibility for Inclusive Interaction
    The website is designed to be accessible for users with hearing disabilities through the extensive use of "aria-label" attributes. These attributes offer detailed descriptions of the site's interactive elements, ensuring that all users, regardless of hearing ability, can fully engage with the content and contribute to the foundation's cause.
